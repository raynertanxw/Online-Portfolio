Life's Lemons

The Game is still in its early stages. As such, music, sound effects, and other audio for the game has not been developed yet. Currently the User Interface design is 75% completed. Gameplay wise, the game is almost done.

Do try out the game using Unity's webGL by clicking the link below. Keep in mind that Unity's WebGL technology is still in preview mode and there may be some quirkiness here and there. Additionally, the web port is inconsistent with its ability to run on various computers and browsers (e.g. Works fine for me on chrome and firefox but an error occurs when I play on Safari. Friends have tested and failed to get it working on their chrome browsers.) The game is built for standalone Linux, Mac, and PC devices. A development build will be made available for download soon.
