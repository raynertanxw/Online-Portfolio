envlight

Envlight is a match-three game with trivia questions about the environment and product marketing messages by the client which I developed this game for. Envlight was made for a Junior Achievement company: Envlight, an eco-friendly company that specialises in using recycled materials that have been collected and processed, thus enabling them to be reused to produce innovative recycled products.

Envlight was set up from a competition called Junior Achievement that puts participants through a programme that enables them to experience running a company. I was part of the Envlight team that won first place in the Singapore round in 2015.

This game was made in a week to help Envlight stand out from the competition in the Asia Pacific round. It was built upon the source code from a Ray Wenderlich tutorial as suggested by my lecturer.
